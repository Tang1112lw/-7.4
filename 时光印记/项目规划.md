# 时光印记 · 家庭成长记录平台 — 项目规划

## 技术栈
- **后端**：Python / FastAPI / SQLAlchemy / Jinja2
- **数据库**：MySQL
- **前端**：原生 HTML + Bootstrap5 最简样式（不做 SPA）
- **部署**：Gunicorn + Nginx + Systemd + Crontab
- **其他**：openpyxl（Excel 导出）

## 核心模块（5 个，聚焦后端技能）

### 1. 用户系统
- 注册 / 登录 / 登出
- 密码哈希存储
- 角色：管理员 / 普通用户

### 2. 家庭成员管理
- 增删改查家庭成员的姓名、关系、生日
- 成员与用户关联（谁创建了谁）

### 3. 成长故事
- 发布标题 + 内容 + 关联成员
- 故事列表（分页）、编辑、删除
- 按成员筛选

### 4. 图片上传
- 上传图片关联到成员或故事
- 文件存储、格式校验

### 5. Excel 导出
- 导出某个成员的年度成长数据为 .xlsx
- 包含故事统计、时间线

### 6. Linux 部署
- Gunicorn + Nginx 反向代理
- Systemd 设置开机自启
- Crontab 数据库每日备份脚本

---

## 不做的
- 复杂前端 UI 美化
- 富文本编辑器
- 标签/分类多维度检索
- 前端路由 / SPA

---

## 开发环境
- 虚拟环境：`venv/`
- 启动命令：`uvicorn app.main:app --reload`
- 数据库：本地 MySQL `timeprint`
